Stockathon

A MEAN stack app that tells you historical stock data as well as stock values for the day

Learn More
 

↓

Github Repository Link

Technical Details

  1. This web application retrieves stock market data from the Alpha Vantage API
  2. The graph that displays monthly stock data was created using the ng2-charts library
  3. General application layout created using the clarity seed HTML/CSS framework
  4. Created a database so users can register, log in, and save specific stocks they want to track. That way, when they log in again, they'll see the most up to date information for the stocks that they care about
